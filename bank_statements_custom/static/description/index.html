Bank Statements

Import your bank statements, manage your payments.

Relate your bank statements to the corresponding invoices.

Statement file content can be imported only if the file corresponds to the selected bank, assuming that the file extension is the same as the bank ID (this condition can be easily changed). Initially, the module works with the statements from only one bank, but it can be upgraded to work for different banks depending on their specifications.

Register Payments

After importing, the content of the statement is listed in tree view. The balance of each invoice that is related to the statement is updated and its state is set as 'paid' if so.

Set Invoices Manually

Users can manually set the invoices that correspond to each payment of the statement. Also, if a wrong invoice is set to any payment, there is always the possibility to change the invoice.
